Logout Requires Server-Side

The original admin_logout.php cleared the admin_id cookie and redirected to ../admin/login.php. In static HTML, cookies cannot be reliably cleared or redirected without JavaScript and a server.

Suggested replacement:

  1. Remove the admin_id cookie via JavaScript if it was set on the same domain.
  2. Redirect users to ../admin/login.html.
<script>
// Example only; adjust cookie name/path as needed
// Deletes cookie by setting expiry in the past
function deleteCookie(name) {
  document.cookie = name + "=; expires=Thu, 01 Jan 1970 00:00:00 GMT; path=/";
}

deleteCookie('admin_id');
window.location.href = '../admin/login.html';
</script>
