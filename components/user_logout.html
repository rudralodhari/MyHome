Logout Requires Server-Side

The original user_logout.php cleared the user_id cookie and redirected to ../home.php. In static HTML, cookies cannot be reliably cleared or redirected without JavaScript and a server.

Suggested replacement:

  1. Remove the user_id cookie via JavaScript if it was set on the same domain.
  2. Redirect users to ../home.html.
<script>
// Example only; adjust cookie name/path as needed
function deleteCookie(name) {
  document.cookie = name + "=; expires=Thu, 01 Jan 1970 00:00:00 GMT; path=/";
}

deleteCookie('user_id');
window.location.href = '../home.html';
</script>
